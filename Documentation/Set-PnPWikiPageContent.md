# Set-PnPWikiPageContent

## SYNOPSIS

## SYNTAX 

### 
```powershell
Set-PnPWikiPageContent [-Content <String>]
                       [-Path <String>]
                       [-ServerRelativePageUrl <String>]
                       [-Web <WebPipeBind>]
                       [-Connection <SPOnlineConnection>]
```
