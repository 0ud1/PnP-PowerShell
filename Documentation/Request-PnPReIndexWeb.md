# Request-PnPReIndexWeb

## SYNOPSIS
Marks the web for full indexing during the next incremental crawl

## SYNTAX 

### 
```powershell
Request-PnPReIndexWeb [-Web <WebPipeBind>]
                      [-Connection <SPOnlineConnection>]
```
