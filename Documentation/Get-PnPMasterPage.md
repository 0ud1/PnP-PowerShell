# Get-PnPMasterPage

## SYNOPSIS
Returns the URLs of the default Master Page and the custom Master Page.

## SYNTAX 

### 
```powershell
Get-PnPMasterPage [-Web <WebPipeBind>]
                  [-Connection <SPOnlineConnection>]
```

