# Remove-PnPTermGroup

## SYNOPSIS
Removes a taxonomy term group and all its containing termsets

## SYNTAX 

### 
```powershell
Remove-PnPTermGroup [-GroupName <String>]
                    [-TermStoreName <String>]
                    [-Force [<SwitchParameter>]]
                    [-Connection <SPOnlineConnection>]
```
